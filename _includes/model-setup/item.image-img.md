<p class="b20" markdown="1">
An image URL. It will be appended to the global <code>rootMediaUrl</code> and the [canvas](#){:data-toc="canvas"} <code>mediaUrl</code> if they are defined.
</p>

{% highlight javascript %}
{
	img: "logo.png"
}
{% endhighlight %}

{% highlight javascript %}
{
	img: "url(logo.png)"
}
{% endhighlight %}
